This code comes with no warrenty.

Use: 

add any dcm files you wish to convert into the "input" folder. 
then open the project directory with cmd

you can RUN with
"cargo run"

you can BUILD an exe with
"cargo build --release"
this will create the target/release directory
you'll need to add your own "input" and "output" folders here before you run the exe

the new .exr files will be saved to the "output" folder
